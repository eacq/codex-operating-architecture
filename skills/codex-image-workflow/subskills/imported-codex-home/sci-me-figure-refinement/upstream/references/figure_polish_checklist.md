# Figure Polish Checklist

## Extraction

- Extract from the original Word/PDF/thesis source at the highest available resolution.
- Prefer original data or vector source over screenshot.
- Track which figure came from which source so later edits remain explainable.

## Repair

- Repair incomplete borders before resizing.
- Remove extra white margins but keep enough caption separation.
- Normalize line weights across subfigures.
- Improve contrast without changing measured data.
- Do not crop away scale bars, legends, coordinate axes, or necessary boundary lines.

## Redrawing

- Redraw workflow diagrams when screenshots contain too much software UI.
- Redraw algorithm schematics around the actual computation steps.
- Use arrows only for process direction or geometric correspondence.
- Keep text short enough to remain readable in one column.

## Mechanical-Engineering Specifics

- For machining figures, distinguish tool, workpiece, contact region, coordinate system, and motion direction.
- For grinding/screw-rotor figures, keep rotor profile, wheel profile, installation parameters, and contact/interference indicators visually separate.
- For verification figures, show condition, metric, and comparison baseline.

## Final Insert

- Insert at the intended final Word size.
- Render the page and inspect the actual size, not the source image alone.
- Re-check caption, cross-reference, and paragraph interpretation after insertion.
