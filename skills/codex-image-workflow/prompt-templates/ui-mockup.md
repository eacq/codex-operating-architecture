# UI Mockup Prompt

Create a polished UI mockup for {{app_or_feature}}.

User workflow: {{workflow}}
Platform: {{platform}}
Visual tone: {{tone}}
Aspect ratio: {{aspect_ratio}}

Visual requirements:

- Show the primary working screen, not a marketing page.
- Use dense but readable controls appropriate to the domain.
- Include realistic states, navigation, and data examples.
- Keep typography consistent; make labels short and legible.
- Use icons for common actions and avoid decorative cards inside cards.

Do not include:

- Placeholder lorem ipsum, nonsensical charts, clipped controls, or overlapping
  text.
