# Frontend Hero Prompt

Create a high-quality hero image for {{product_or_project}}.

Domain: {{domain}}
Target user: {{audience}}
Mood: {{mood}}
Aspect ratio: {{aspect_ratio}}

Visual requirements:

- Make the product, place, object, or workflow immediately recognizable.
- Use real-feeling material detail rather than abstract gradients or blobs.
- Keep the composition usable behind or beside concise web copy.
- Avoid crowded text; if text is needed, keep it minimal and readable.
- Leave natural negative space for layout adaptation.

Do not include:

- Generic stock-photo staging, fake brand marks, watermarks, or unreadable UI.
