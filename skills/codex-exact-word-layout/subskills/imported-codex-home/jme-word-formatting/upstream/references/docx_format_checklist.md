# DOCX Format Checklist

## Before Editing

- Identify the latest manuscript by modification time and user instruction.
- Create a backup with a meaningful suffix.
- Locate template/sample documents and extract styles rather than guessing.

## Citations And References

- Body citations are superscript bracketed numbers.
- Reference-list labels remain normal baseline text.
- Reference entries use hanging indent and consistent line spacing.
- Chinese and English bilingual reference punctuation follows the supplied sample.
- Cross-paper comparisons do not create uncited claims.

## Equations

- Displayed equations are Word equations, centered or formatted according to template.
- Equation numbers are aligned consistently and not placed by ad hoc tables unless the template requires it.
- Inline formula symbols are Word equation/math format where requested.
- Subscripts/superscripts are checked in formulas and surrounding formula mentions.

## Figures And Tables

- Use original images as the first source for refinement.
- Preserve borders when the original figure depends on them.
- Prefer single-column width for readable visuals.
- Tables use consistent header shading, border weight, column alignment, and compact but readable cell padding.
- Captions use journal-style numbering and are close to the object they describe.

## Final QA

- Run structural audits where available.
- Export with Word or the most faithful renderer available.
- Inspect all rendered pages or a full contact sheet plus critical pages.
- Re-check pages containing edited citations, references, equations, tables, and figures.
