# Image Workflow Module

Owns visual selection, generated or licensed image provenance, prompt iteration, post-generation image feedback, [[Image Hosting and Cleanup Workflow]], and recovery manifests. It supports [[Knowledge System Module]] and [[Scientific Figure Workflow]]. Prefer Mermaid when the visual is structural and does not require a bitmap; when a bitmap is useful, finish with provenance, optional hosting, quarantine, and manifest-backed recovery.
