# Exact Word Layout Module

`codex-exact-word-layout` owns the reusable workflow for exact-format DOCX layout repair.

## Ownership

- Authoritative DOCX selection.
- Versioned archive and manifest discipline.
- Targeted OOXML or Word COM layout edits.
- Package-invariant snapshots and narrow semantic XML exceptions.
- Render-first visual QA gates.
- Promotion boundary between generic layout behavior and project-specific template facts.

## Non-Ownership

- General document creation.
- Ordinary comments, redlines, and conventional style cleanup owned by `documents`.
- Journal-specific typography and citation style.
- Scientific prose rewriting.
- Figure data generation or source-evidence alteration.
