# Provider Capability Boundary

OpenAI-compatible chat and embeddings are interface capabilities, not universal substitutes. [[Literature Discovery Workflow]], [[Scientific Figure Workflow]], and [[Zotero Reference Workflow]] retain domain providers when their APIs or media capabilities differ.
