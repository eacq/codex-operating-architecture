# Inbox

Temporary captures land here. Process each item into one atomic note, merge it into an existing note, or discard it. Do not let candidate notes become verified knowledge without evidence. Quicker can call `quick_capture.ps1` with a title and selected or clipboard text.

Promoted notes should link to [[Knowledge System Module]] and the relevant workflow or concept.
