# Network Learning: JuliusBrussee/caveman selective integration

## Trigger and outcome contract

The user authorized installation and integration of Caveman. The observable
goal is lower routine response overhead without weakening the quality, safety,
privacy, evidence, or human-readable reporting floor of the global experience
system.

## Evidence and comparison

- Source: https://github.com/JuliusBrussee/caveman (reviewed 2026-07-18).
- Upstream offers a core skill, optional command skills, a cross-agent
  installer, hooks, a status display, local token statistics, memory-file
  rewriting, optional MCP-description compression, and subagent helpers.
- Existing `codex-self-evolution` resource-economy already owns model/context,
  tool, time, and user-attention decisions. Existing lifecycle, error-feedback,
  Git, and content-production owners already require clear evidence and
  reader-facing documentation.

## Decision

Install only `skills/caveman` into the local Codex skill catalog. Integrate its
useful principle into `resource-economy`: brevity is an opt-in presentation
choice, not a reduction in reasoning, verification, or authority checks.
Preserve user language and exact technical artifacts. Force clarity back on
for safety, irreversible actions, ambiguous sequences, and formal reports.

## Rejected or deferred

- Do not run the upstream all-agent installer, hooks, statusline integration,
  session-log statistics, memory rewriting, MCP middleware, or cavecrew
  subagents.
- Do not automatically activate compressed prose for Chinese-primary candidate
  reports, release notes, error reports, or other reader-facing documents.
- Do not accept the upstream percentage claim as a local cost-saving result;
  it is output-token focused and must be measured locally before promotion.

## Verification and invalidation

Verify the installed local skill exists and validate the architecture
repository. Revisit if explicit brevity requests repeatedly produce ambiguous
or incomplete handoffs, if installed skill instructions conflict with current
authority/clarity gates, or if local measurements fail to show a useful benefit.
